Formulários

Um formulário trabalha com 2 tipos de metodos que são "GET" ou "POST" e eles são indicados com os atributos method="". A diferença entre o GET e o POST é que o GET os dados colocados no formulário aparecerão na barra de endereço e o POST será ocultado e o usuário não verá nada.

Value

Em alguns inputs, como o checkbox, você deve por um value="" para indetificar individualmente cada umas das opções. O Value em outras palavras seria um valor real de um determinado imput, na maioria das vezes ele conterá o valor real de tal imput e na maioria das vezes ele fica imbutido e o usuário não enxerga esse valor.



Name

Não esqueça de por o name="" para indicar qual o elemento é que o botão de submit está se referindo. Se você não por o atributo name="" o elemento não irá ser enviado para o backend já que ele não sabe o que é aquele elemento.

HTML 5

Para criar um formulário usando um seletor de COR, user o input type="color" OBS: Alguns navegadores não funciona ainda esse recurso do HTML5

Depois você pode usar o seletor de datas com o type="date" e pode também especificar um range entre a data máximo e minima usando max="" e min=""

Colocando um type="datatime-local" você também pode adicionar um relógio ao calendário.

Para por uma entrada de arquivos use o input type="file" PARA ARQUIVOS O METHOD DO FORMULÁRIO DEVE SER DO TIPO POST

Para criar um campo de entrada exclusiva para numeros usa-se type="number" Você também pode aplicar um max="" e min="" para estipular os numeros máximos e minimus que serão aceitos.

Agora se você quiser criar uma barra de rolagem horizontal para que o usuario escolha um range, você usará type="range" também é possível usar min e max.

Para aceitar numéros telefónicos, usa-se type="tel". Na pratica ele serve para que no telefone o navegador abra o teclado numerico do telefone para o usuário.

Outro input já conhecido é o type="email" que serve para que o html possa aceitar ou não uma determinada entrada. Se ela for uma entrada do tipo email ele aceitará, se não, não permitirá que o usuário envie o formulário.
