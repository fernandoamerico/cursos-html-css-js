Box Model

Sempre que você usar margens, paddings e bordas você deve observar os tamanhos dos elementos para que eles não sejam deformados ou percam seus tamanhos por ter uma margem muito grande ou um padding mal calculado


Usando Outline

Quando usamos bordas, margens e paddins, esses valores interagem com esses mesmos valores de outros elementos, por exemplo, a margem de uma div pode interagir com a margem de outra div, podemos controlar um mas cusando "outline" esse tipo de intarassão não acontecerá, porque o outline flutua em cima dos outros elementos do html.

box-sizing: border-box

Agora imagine que você precisa de um elemento que tenha uma borda de um tamanho X mas que ele não pode ficar maior do que o seu temanho original, o que podemos fazer para resolver esse problema? Usammos a propriedade box-sizing: border-box; Com essa propriedade, as bordas de um elemento não irá crescer mas para do elemento, aumentando o seu tamanho, ela irá diminuir o tamanho original do elemento e crescerar para dentro, assim deixando o tamanho de fora do elemento sem modificação.
