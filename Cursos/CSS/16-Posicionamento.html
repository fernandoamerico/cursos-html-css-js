Posicionamento

Para trabalhar posicionamentos com CSS você pode usar a propriedade position, onde usando alguns valores específicos, você pode configurar como o elemento irá se comportar referente a sua posição.

Static e Relative

Por padrão as divs tem a propriedade static, em relação ao atributo, ou seja, ela irá ficar parado em sua posição inicial.

div normal
static

Depois temos o atributo relative, que usando valores específico, como left, top e outros, você pode alterar a posição inicial da div.

div normal
Relative {
top: 20px;
left: 15px;
}

Fixed

Usando fixed a div irá sempre ficar fixo no lugar indicando usando a janela com referenciais


Absolute

A propriedade absolute funciona no primeiro momento como o static, mas ao aplicar um top, left, right o bottom, ela irá funcionar como a propriedade relative, mas usando agora o elemento pai como referencia, não a posição inicial como é o caso do relative. Se ela não tiver uma div pai, ela usará o body como pai. OBS>>> O ELEMENTO PAI NÃO PODE SER DO TIPO STETIC, SE NÃO, NÃO FUNCIONARÁ!

div normal
absolute {
left: 0px;
}





OBS: ele passa a flutuar na tela e não tem interação com outros elementos do tipo block.


Sticky

Com essa propriedade, você irá definir ao HTML que tal elemento, não pode ficar com menos do que uma certa quantidade de pixel, referente ao top, left, right ou bottom, assim sendo, se você define por exemplo, top: 0px, a div não irá sair da tela, não importantdo a quantidade de vezes que você gire a tela com o scroll, pois ela está configurada, para não permitir que o top de tal elemento, fique inferior a 0(zero).

sticky
{ top: 100px;
bottom: 5px;
}
