Overflow

O Overflow serve para trabalhar com conteúdos que são maiores do que seus elementos. Veja os Exemplos a baixo.

overflow: hidden

Usando o atributo hidden ao overflow, observe que o texto que é o conteúdo de nossa div é maior que a própria div, mas ela mantem o seu tamanho pré estabelecido e com o atributo hidden, o texto que passa desse tamanho, passa a ficar oculto.

Auto

Uma outra forma de trabalhar com o overflow é usando o atributo auto, onde você deixa que o próprio html aplique o scroll de acordo com o tamanho do conteúdo dentro do elemento.
